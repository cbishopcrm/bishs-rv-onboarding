# Bish's RV Onboarding Guide

Static onboarding site for Bish's RV sales team on TheCRM platform.

- **Live site:** https://bishs-rv-onboarding.vercel.app
- **Stack:** Single static `index.html`, deployed on Vercel.

## Deployment

Pushes to `main` auto-deploy to production via the Vercel GitHub integration.

## Video assets

Training videos are embedded from a shared Google Drive folder. For the embeds to work for viewers, the Drive files must be shared as **Anyone with the link — Viewer**.
